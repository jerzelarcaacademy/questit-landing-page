# Project: QuestIt

A landing page for QuestIt — a gamified task marketplace where you post a quest, set a bounty, and a nearby Hunter gets it done today. Built for busy professionals, small business owners, and skilled freelancers in the Philippines.

The design should feel bold, confident, and game-inspired — like an RPG quest board meets a modern product page.

## Always

- Invoke the `frontend-design` skill before writing any front-end code.
- Reference `brand_assets/` for the logo, colors, and brand guidelines. Brand colors and tagline are in `brand_assets/brand.md`. The logo SVG is at `brand_assets/logo.svg`.
- Keep the aesthetic dark, sharp, and intentional — gold (#f5c842) and purple (#7b6ef6) on near-black (#0a0a0f).
- Use real copy that reflects the QuestIt brand voice: direct, action-oriented, slightly playful.

## Stack

- HTML, CSS, vanilla JavaScript. No framework.
- Fonts: Cinzel (headings) + Inter (body) via Google Fonts.
- Deploy on Vercel via GitHub auto-deploy.

## Never

- Use placeholder Lorem Ipsum text.
- Ship designs that look "AI vibe coded": generic gradient blobs, fake stat counters, badge-and-pill hero sections with no real content, or stock-photo-style imagery.
- Use generic SaaS design patterns (pricing table grids, feature icon rows, testimonial carousels) without adapting them to the QuestIt brand.
- Modify files in `brand_assets/`.
